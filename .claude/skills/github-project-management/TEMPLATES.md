## 使用例

### 新機能追加の場合

親Issue:
```
feat: ユーザー認証機能
```

サブIssue:
```
feat(auth): データベーススキーマ追加
feat(auth): 認証API実装
feat(auth): フロントエンド連携
feat(auth): テスト追加
```

### バグ修正の場合

親Issue:
```
fix: ログイン時のセッション管理不具合
```

サブIssue:
```
fix(session): セッションタイムアウト修正
fix(session): トークン更新ロジック修正
```
